# Proyek Klasifikasi - Dataset Iris

Proyek ini bertujuan untuk memprediksi spesies bunga Iris berdasarkan 4 fitur utama.

## Isi Proyek
- Exploratory Data Analysis (EDA)
- Preprocessing (Scaling, Train-Test Split)
- Model Klasifikasi:
  - Logistic Regression
  - Decision Tree + Hyperparameter Tuning
- Evaluasi menggunakan Confusion Matrix, Accuracy, Precision, Recall, F1-score, ROC-AUC

## File Penting
- `iris_classification_project.py` : Script utama proyek
- `Laporan_Klasifikasi_Iris.docx` : Laporan maksimal 3 halaman
- `joblib` Model disimpan otomatis setelah training

## Cara Menjalankan
```bash
pip install -r requirements.txt
python iris_classification_project.py
```
